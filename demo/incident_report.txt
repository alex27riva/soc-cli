1. EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
On 2026-04-19 at approximately 02:14 UTC, automated alerting detected anomalous
outbound traffic originating from a workstation in the Finance department
(FINWS-047). Investigation revealed a multi-stage compromise involving an initial
phishing email, a PowerShell-based loader, and deployment of a remote access
trojan (RAT). The attacker exfiltrated financial documents and subsequently
attempted to extort the organization via a ransom note demanding Bitcoin payment.
The affected host has been isolated and remediation is ongoing.

10. RECOMMENDATIONS
--------------------------------------------------------------------------------
  1. Disable Office macros by default via GPO; allow only signed macros.
  2. Enable PowerShell Constrained Language Mode and Script Block Logging.
  3. Deploy application allowlisting on Finance workstations.
  4. Implement DNS-layer filtering (e.g. RPZ) to block known C2 domains.
  5. Monitor for double-extension file execution at the EDR level.
  6. Add Bitcoin address pattern detection to DLP rules.
  7. Enforce DMARC/DKIM/SPF on all inbound mail to reduce spoofed senders.
